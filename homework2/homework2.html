Part 1: Dataset & Distributions

A dataset is a structured collection of data organized in rows (records) and columns (variables). Each variable represents a measurable attribute. Variables can be:

  • Categorical: values in categories (e.g., Status: Success / Failed)
  • Numerical: measurable numbers (e.g., Login Attempts)

In cybersecurity, datasets help analyze user behavior, detect anomalies, or track login attempts.
