Homework 10 — Discrete approximation of Poisson counting process

Simulate events uniformly in time at rate λ using n subintervals and compare Binomial → Poisson approximation.

Setup

Divide time interval \([0,T]\) into \(n\) equal subintervals. In each subinterval generate an event with probability \(\lambda T / n\). The number of events in \([0,T]\) is Binomial\((n,\lambda T/n)\) and for large \(n\) this converges in distribution to Poisson\((\lambda T)\).

\(K_n \sim \mathrm{Binomial}\!\left(n,\frac{\lambda T}{n}\right) \quad,\quad K_n \xrightarrow[n\to\infty]{d} \mathrm{Poisson}(\lambda T).\)
