1. Introduction

In this homework, we use the Euler–Maruyama method, previously introduced in earlier assignments, to simulate a Wiener process (Brownian Motion), one of the most fundamental objects in modern probability theory. A Wiener process is a continuous-time stochastic process used to model randomness in physics, finance, biology, and network traffic.

We also optionally extend the simulator to allow simulation of general stochastic differential equations (SDEs) of the form: 

dXₜ = μ(Xₜ, t) dt + σ(Xₜ, t) dWₜ

2. What is a Wiener Process?

A Wiener process W(t) is a stochastic process defined by:

  • W(0) = 0
  • Independent increments
  • W(t) − W(s) ~ N(0, t − s) for t > s
  • Continuous paths almost surely

It represents the mathematical model of continuous random movement and is the core ingredient of many stochastic models.

3. Euler–Maruyama Method (Definition)

To simulate an SDE numerically, we discretize time into n small steps of size Δt. For a general SDE: 

dXₜ = μ(Xₜ, t) dt + σ(Xₜ, t) dWₜ
the Euler–Maruyama update rule is: 

X_{k+1} = X_k + μ(X_k, t_k) Δt + σ(X_k, t_k) √Δt · Z_k
where:
  • Z_k ~ Normal(0,1)
  • Δt = T / n

For a Wiener process specifically: 

μ = 0
σ = 1

so the update becomes:   W_{k+1} = W_k + √Δt · Z_k
