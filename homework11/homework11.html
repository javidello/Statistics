Homework 11 — Wiener Process (Brownian Motion)

Task Overview

Simulate a Wiener process using the Euler–Maruyama method. Optionally, extend to general SDEs.

Theory Summary

Wiener Process

  • Initial value: W₀ = 0
  • Independent increments
  • Normally distributed increments: Wₜ − Wₛ ~ N(0, t − s)
  • Continuous paths but nowhere differentiable

Euler–Maruyama Update

W_{n+1} = W_n + √Δt · Zₙ, where Zₙ ~ N(0,1)
